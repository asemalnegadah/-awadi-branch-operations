# نظام تشغيل ورقابة فرع العوادي – عدن

هذا المستودع هو المرجع التنفيذي الرسمي لبناء نظام تشغيل ورقابة فرع العوادي في عدن فقط.

## نطاق النظام

- فرع عدن فقط، دون تعدد فروع.
- واجهة عربية RTL متجاوبة وقابلة للتثبيت كتطبيق PWA.
- قاعدة البيانات الرسمية PostgreSQL.
- فصل SR عن RG في جميع الحركات والتقارير.
- الأرصدة ناتجة عن دفتر حركات، وليست أرقامًا قابلة للتعديل المباشر.
- جميع العمليات الحساسة تمر بمراحل مراجعة واعتماد وتدقيق.

## المرجع الرسمي

سيتم حفظ المواصفة الرئيسية داخل:

`docs/MASTER_SYSTEM_SPECIFICATION.md`

ولا يبدأ بناء أي وحدة تشغيلية قبل توثيق:

1. نطاق النسخة الأولى.
2. دفتر الحركات والأرصدة.
3. ERD وقاموس البيانات.
4. حالات العمليات ومسارات الاعتماد.
5. مصفوفة الصلاحيات.
6. قواعد الخطط والمخاطر.
7. خريطة نقل بيانات النظام الحالي.
8. معايير القبول والاختبارات.

## قواعد حاكمة

1. لا دمج بين SR وRG في حركة واحدة.
2. لا حذف للحركات المالية المعتمدة؛ التصحيح بحركة عكسية.
3. لا ترحيل دون معاينة واعتماد.
4. لا رقم دون مصدر وتاريخ قطع بيانات.
5. لا خطة من بيانات غير معتمدة.
6. لا يعتمد منشئ الحركة عمليته المالية الحساسة بنفسه.
7. لا تحفظ الأسرار أو بيانات العملاء داخل المستودع.
8. كل تعديل حساس يسجل في سجل تدقيق.
